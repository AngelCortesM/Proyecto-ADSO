HTML Básico

En esta sección, exploraremos los conceptos fundamentales del lenguaje HTML.

Etiquetas HTML

HTML utiliza etiquetas para estructurar y dar formato al contenido de una página web. Aquí tienes algunas etiquetas esenciales:

  • <html>: Define el documento HTML.
  • <head>: Contiene información sobre el documento, como el título y enlaces a estilos.
  • <body>: Contiene el contenido principal de la página.
  • <h1>, <h2>, <h3>: Encabezados de diferentes niveles.
  • <p>: Define un párrafo de texto.

Listas

Las listas son una parte importante de HTML:

  • <ul>: Lista no ordenada.
  • <ol>: Lista ordenada.
  • <li>: Elemento de lista.

Enlaces

Los enlaces permiten la navegación entre páginas. Aquí está la etiqueta básica:

<a href="url">Texto del enlace</a>

Imágenes

Para agregar imágenes a tu página, utiliza la etiqueta <img>:

<img src="ruta-de-la-imagen" alt="Descripción de la imagen">

CSS: Estilizando tu Página

En esta sección, exploraremos cómo dar estilo y diseño a tu página web mediante CSS.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que describe la presentación de un documento HTML. Con CSS, puedes controlar el diseño, los colores, las fuentes y otros aspectos visuales de tu página.

Selección de Elementos

Para aplicar estilos, primero seleccionamos los elementos HTML. Puedes seleccionar por etiquetas, clases o identificadores:

body {
            font-family: 'Arial', sans-serif;
            background-color: #f4f4f4;
        }
.destacado {
            color: #ff6600;
            font-weight: bold;
        }

Propiedades y Valores

Cada estilo en CSS se compone de propiedades y valores. Por ejemplo, color: #333;, donde color es la propiedad y #333 es el valor.

Box Model

El modelo de caja en CSS describe cómo se representan y organizan los elementos en la página. Incluye propiedades como margin, padding, border, etc.

Imágenes y Fondos

Controla la presentación de imágenes y fondos con propiedades como background-image y border-radius.

Estilizando con Bootstrap

Descubre cómo utilizar Bootstrap para darle un aspecto moderno y responsivo a tu página web.

Botones Bootstrap

Bootstrap proporciona una variedad de estilos predefinidos para botones. Puedes utilizar clases como btn btn-primary para resaltar un botón o btn btn-outline-dark para un estilo de contorno.

<button type="button" class="btn btn-primary">Botón Principal</button>

Barra de Navegación

La barra de navegación de Bootstrap facilita la creación de menús de navegación elegantes y responsivos. Puedes usar la clase navbar para comenzar.

<nav class="navbar navbar-expand-lg navbar-light bg-light">
            <a class="navbar-brand" href="#">Logo</a>
            <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
                <span class="navbar-toggler-icon"></span>
            </button>
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav">
                    <li class="nav-item active">
                        <a class="nav-link" href="#">Inicio <span class="sr-only">(current)</span></a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#">Acerca de</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#">Servicios</a>
                    </li>
                </ul>
            </div>
        </nav>

Grid System

Bootstrap utiliza un sistema de cuadrícula (grid) para ayudarte a diseñar el diseño de tu página de manera fácil y efectiva.
